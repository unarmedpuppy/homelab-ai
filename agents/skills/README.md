# Skills

Reusable workflow guides for common tasks.

## Available Skills

| Skill | Description |
|-------|-------------|
| [standard-deployment](./standard-deployment/SKILL.md) | Deploy code changes to server |
| [deploy-new-service](./deploy-new-service/SKILL.md) | Set up a new Docker service |
| [troubleshoot-container-failure](./troubleshoot-container-failure/SKILL.md) | Debug container issues |
| [troubleshoot-stuck-downloads](./troubleshoot-stuck-downloads/SKILL.md) | Fix Sonarr/Radarr queue issues |
| [system-health-check](./system-health-check/SKILL.md) | Comprehensive system status |
| [cleanup-disk-space](./cleanup-disk-space/SKILL.md) | Free up disk space |

## How to Use

Skills are markdown guides with step-by-step commands. Read the relevant skill and follow the steps.

## Creating New Skills

1. Create a directory: `mkdir skills/skill-name`
2. Create `SKILL.md` with:
   - When to use
   - Step-by-step commands
   - Common issues/fixes
