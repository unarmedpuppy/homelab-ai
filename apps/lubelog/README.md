# LubeLogger

Self-hosted, web-based vehicle maintenance and fuel mileage tracker.

## Description

LubeLogger helps you manage vehicle maintenance records and monitor fuel consumption, providing a centralized platform for tracking and analyzing vehicle-related data. No more spreadsheets or shoeboxes full of receipts!

## Features

- Vehicle maintenance tracking
- Fuel mileage monitoring
- User-friendly web interface
- Self-hosted and open-source

## Configuration

The service uses LiteDB by default (stored in `./data`). No additional database setup required.
